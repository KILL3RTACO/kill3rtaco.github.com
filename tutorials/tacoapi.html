About

TacoAPI is an API I made to make my plugins easier to manage, but it still available at BukkitDev. The plugin doesn't really do anything besides provide another layer for developers to work with.

Management

The plugin is updated as I see fit. This means that I will mostly update it only to fit the plugins I develop. Other than that, if a feature is suggested, I may or may not implement it.

Before I Continue

Before continuing with this general tutorial, it assumed you know how to make a Bukkit plugin in the first place. This includes making a server to test it on. If you do not, you can always check the BukkitWiki. This also includes learning Java. This is a must, and if you haven't, you'll need to do that first. I'll leave it up to you and Google/YouTube from this point.
