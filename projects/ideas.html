EasyBukkit

This application would be written in Java and would be Open Source. It would allow the server owner to modify the contents of the server.properties file.

Features

  • Easy server management
  • Tabs to sort out different properties
  • Predefined values can be selected easily with drop down menus or checkboxes

PartyChat

This Bukkit plugin would provide a service for players to communicate in a group. The PartyLeader can invite or kick players, as well as setting the join status. If the join status is invite-only, you can only join if invited. If it isn't invite-only (open) then any player can join your party at will.

Features

  • Group chatting
  • Party management
  • Player blocking? (If they talk inside a party you're in you can't see/the message is censored. Maybe use &k?)
