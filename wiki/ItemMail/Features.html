Mail

ItemMail's key feature is sending mail. You can easily send mail to other players. Mail is stored in a mailbox which can be viewed in pages. Mail can be opened by using the command /im open [#] with [#] being the corresponding number in the mailbox.

Requests

Requests are another feature in ItemMail. A request is when you want another player player to send you items. If accepted, the items will be sent to you in the form of mail. Requests can be accepted with /im accept [#] and declined with /im decline [#], with [#] being the corresponding nu,ber in the mailbox.

Mailbox System

Mail and Requests are stored in separate mailboxes. When viewed, each item in the mailbox has a certain index, or number, to the left of it. Brief information is displayed next to each mailbox item, like how much and of what item it is.
