Commands

Note: A subcommand is the second 'word' when writing out the command. For instance, if the command you write out was /im send kill3rtaco diamonds then the subcommand would be send

Subcommand Parameters Description
Mail
send <player> [item:damage] [amount] Sends mail to another player.
open [#] Opens the specified mail. The last mail sent to you will be opened if unspecified.
delete [#/*] Deletes the specified mail. The last mail sent to you will be deleted if unspecified. If an asterisk is used, all mail will be deleted
Requests
request <player> [item] [amount]
accept [#] Accepts the specified request, or the last one if undefined.
decline [#/*] Declines the specified request or the last one if undefined. If an asterisk is used, then all requests will be declined.
