ItemMail

ItemMail is a helpful designed for player trade. It provides:

TacoAPI

TacoAPI is an API I made for myself, but is still available and open source for developers to use. It provides:

  • Command handling functionality based off of DeityAPI, but enhanced a little bit
  • Various other helpers like chat formatting
